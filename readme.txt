Задание по курсу функционального программирования написано на scheme в среде racket.

Имеются директории с описание двух частей задания (обе части реализованы в одной программе),
директория с данными (модели и тексты для обучения), директория с отчетом о проделанном задании и
директория с исходным кодом обучающей прогаммы и самого доктора.
